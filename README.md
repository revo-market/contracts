# contracts
Revo smart contracts.

## Installation
```
yarn
```

## Deploying to Alfajores
1. Fill out `scripts/deploy.ts` with the desired parameters
2. Compile contracts with `yarn compile-contracts`
3. Deploy with `yarn deploy-contracts-alfajores`
